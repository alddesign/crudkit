Holds a set of usefull methods.

In most cases the static methods are all you need. An instance of this Object is only needed by Crudkit itself.

addData()

In the Update From there is the special "delete" checkbox for blob/image fields.

If the user doesnt check this checkbox, doesnt provide a file, and there is already data in the DB, we have to ensure not to delete the existing data. This will bypass the above maxtrix with all the logic of null,empty,value...
